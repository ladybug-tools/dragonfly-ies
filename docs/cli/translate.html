translate¶

dragonfly-ies translate¶

Commands for translating Dragonfly JSON files to IES files.

dragonfly-ies translate [OPTIONS] COMMAND [ARGS]...

model-to-gem¶

Translate a Dragonfly Model JSON file to an IES GEM file.

Args:
model_json: Full path to a Model JSON file (DFJSON) or a Model pkl (DFpkl) file.
dragonfly-ies translate model-to-gem [OPTIONS] MODEL_JSON

Options

-n, --name <name>¶

Name of the output file.

Default

model

-f, --folder <folder>¶

Path to target folder.

Default

.

Arguments

MODEL_JSON¶

Required argument
